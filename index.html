Trajectory Generation with Flow-Based Energy Models [Code]

  1. Developed a novel multi-modal flow-based model for trajectory generation that transforms random noise into diverse, realistic paths using continuous flow matching, resulting in high-quality trajectory synthesis with multiple viable solutions for each scenario.
  2. Built an energy-based model on top of the multi-modal flow matching network to quantify the generated trajectories based on energy.

Future Depth Map Prediction Using 3D CNNs [Code]

  1. Collected object flight demonstrations in the MuJoCo simulator with varying velocity and direction attributes for a diverse set of objects. Designed a 3D CNN network to process depth maps from the first T frames and predict depth maps for T+N frames, enabling fast and efficient future scene prediction.
  2. Improved the network's performance using attention mechanisms, achieving a loss of 1e-3.
